# Price Cards

A small front-end practice project created as part of my Web Fundamentals learning path.

## Overview

This project recreates a simple pricing-card interface using HTML and CSS. It was built to practice turning a basic layout into a clear, responsive visual component.

## What I practiced

- Structuring page content with semantic HTML.
- Styling layouts and visual hierarchy with CSS.
- Working with spacing, color, typography, borders, and reusable card patterns.
- Applying front-end fundamentals in a small, focused project.

## Technologies

- HTML
- CSS

## Context

This repository is a learning project. It represents practical work from my Web Fundamentals course and is kept as part of my technical learning portfolio.
